# SCT_DS_4
Project Overview

This project focuses on analyzing road accident data to identify patterns, trends, and key factors contributing to accidents.
Using Python and data analysis libraries, the project explores accident statistics such as frequency, severity, time-based trends, and contributing conditions to gain actionable insights.<br>
The analysis is implemented in a Jupyter Notebook, making it easy to understand and reproduce.

Objectives

To analyze road accident data using Python<br>
To identify major factors causing road accidents<br>
To study trends based on time, location, or accident severity<br>
To visualize accident patterns for better understanding
